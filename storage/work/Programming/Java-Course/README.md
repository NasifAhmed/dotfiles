## How to Use This Repository

1. **Follow the numbered order**: Start with 01-Basics and progress through each section
2. **Read the README** in each folder for specific instructions
3. **Compile and run**: Use `javac FileName.java` to compile and `java FileName` to run
4. **Practice**: Modify the code and experiment with different approaches

## Prerequisites

- Java Development Kit (JDK) installed
- Basic understanding of programming concepts (helpful but not required)

## Getting Started

```bash
# Navigate to any lesson folder
cd Java-Course/01-Basics

# Compile a Java file
javac Hello.java

# Run the compiled program
java Hello
```
