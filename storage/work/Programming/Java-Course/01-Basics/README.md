## How to Get Started

1. **Set up your environment**: Make sure you have Java JDK installed
2. **Navigate to this folder**:
   ```bash
   cd Java-Course/01-Basics
   ```

3. **Compile your first program**:
   ```bash
   javac Hello.java
   ```

4. **Run your program**:
   ```bash
   java Hello
   ```

## Key Concepts

### The Main Method
Every Java program needs a main method:
```java
public static void main(String[] args) {
    // Your code goes here
}
```

### Printing to Console
```java
System.out.println("This prints on a new line");
System.out.print("This stays on the same line");
```

### Java Program Structure
- **Class**: Every Java program is inside a class
- **Main method**: The entry point of the program
- **Statements**: Instructions that perform actions
