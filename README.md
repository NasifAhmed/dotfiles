# dotfiles
## My linux distro dotfiles

*Find config files for each programs by navigating to their directory. The extra .config folders inbetween are for automating stuff with gnu-stow. Dont be confused. The files are there.*




### Current Setup : 

* **Distro** : Arch
* **Window Manager** : dwm 
* **Launcher** : d-menu
* **Locker** : slock
* **Status** : dwm-status (Self Manged)
* **Terminal** : Alacritty 
* **Compositor** : Picom
